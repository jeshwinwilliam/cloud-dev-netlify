Netlify + Neon Visitors App

This site connects a Netlify frontend to a Neon PostgreSQL database through Netlify serverless functions. It supports POST, GET, and DELETE operations.

About This Assignment

The homepage stays visible as a normal static site, and it also includes a direct download link for get-visitors.js. This matches the updated homework requirement to show index.html while also providing a downloadable link for the API file.

API Endpoints

  • GET /.netlify/functions/get-visitors
  • POST /.netlify/functions/add-visitor
  • DELETE /.netlify/functions/delete-visitor?id=VISITOR_ID

After deployment, replace the relative paths with your full Netlify site URL when you submit.

Database Table

Run the SQL below once inside Neon SQL Editor:

CREATE TABLE IF NOT EXISTS visitors (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    visited_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
